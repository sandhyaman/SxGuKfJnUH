# 项目简介

## 核心功能
- 数据清洗
- 数据预处理
- 数据统计
- 数据可视化
- 研究结论

## 适用场景
- 考研考生
- 教育机构
- 教育数据分析研究

- 使用pyecharts、pandas、numpy、matplotlib等Python库进行数据分析与可视化
- 数据集来源于和鲸社区，保证数据来源的可靠性和权威性
- 提供完整的数据集、源码和PPT，方便用户进行学习和应用
